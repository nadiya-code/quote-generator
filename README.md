Quote-Verse
A modern, responsive Single Page Application (SPA) designed to curate and explore timeless wisdom. This project focuses on efficient DOM manipulation and persistent data storage using vanilla web technologies.
Features
 Category-Based Filtering: Browse curated quotes through specialized categories like Philosophy, Science, Literature, and Courage.
 Persistent Favorites: Integrated a "Save" system using LocalStorage to allow users to build a personal collection that persists across browser sessions.
 Dynamic UI: Engineered a seamless SPA experience with section-based navigation (Home, Explore, Authors, Saved).
 Real-time Search:A responsive search bar that filters through authors and quote text instantly.
 Utility Tools:Built-in "One-Click Copy" functionality using the Clipboard API and a "New Quote" generator for a dynamic experience.
Tech Stack
 Core: HTML5, Vanilla JavaScript (ES6+).
 Styling:Tailwind CSS (Mobile-first, responsive design).
 Icons: FontAwesome.
 Persistence:Web Storage API (LocalStorage).
Project Context
I developed this project while pursuing my B.Tech in Artificial Intelligence & Machine Learning. It served as a practical application of the web fundamentals I studied at Uttaranchal University and through various certifications.
Key Learning Outcomes:
 DOM Manipulation:Dynamically creating and rendering quote cards based on user interaction.
 State Management:Handling active UI states and navigation without external frameworks.
 Data Structures:Managing and filtering through a library of 100+ quote objects.
Project Status: Completed the Vanilla JS version. Currently refactoring the architecture to React.js to enhance component modularity.
